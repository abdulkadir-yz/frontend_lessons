# Frontend Lessons

A personal learning repository documenting my frontend development journey through WBS Coding School Bootcamp. This repo covers core web technologies from the ground up — structured, topic by topic.

## What's Inside

| Folder | Topics |
|---|---|
| `01_html` | Semantic HTML, forms, accessibility, page structure |
| `02_css` | Selectors, flexbox, grid, responsive design, animations |
| `03_javascript` | ES6+, DOM manipulation, async/await, fetch API |
| `04_react` | Components, hooks, state management, routing |

## Purpose

This is not a tutorial copy-paste repo. Every file here represents something I actively learned, practiced, and understood. The goal is to build a solid frontend foundation alongside .NET backend development.

## About Me

Software Engineering student at WBS Coding School with a background in IT consulting and DevOps. Working towards becoming a full-stack developer with a focus on .NET backend and modern frontend technologies.
